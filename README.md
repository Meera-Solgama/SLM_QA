# Gujarati QA System

An AI-powered Question Answering system for Gujarati language with question generation capabilities.

## Features
- Generate questions from Gujarati text
- Extract answers from context
- Classify question types (9 categories)
- Determine difficulty levels (3 levels)
- ~80% accuracy on classification
- User-friendly Streamlit interface

## Installation

### Windows:
1. Clone the repository
2. Run `setup.bat`
3. Run `run.bat`

### Manual Installation:
```bash
pip install -r requirements.txt
streamlit run app.py
